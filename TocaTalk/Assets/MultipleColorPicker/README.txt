# MultipleColorPicker
This asset allows you to display multiple color pickers at all times.

## Usage
1. Select Multiple Color Picker from the Tools tab to open the window.
2. Select a game object in the Hierarchy.
3. Select the color property you want to edit from the Target pop-up.

## Features 
* You can have multiple color picker windows open at the same time.  
* It supports all color properties displayed in the inspector. 
* You can lock the color properties to be edited by checking the Lock checkbox.
